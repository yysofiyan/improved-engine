# Penjelasan Perubahan Format NIM

## Format NIM Baru
`TTFFPPJSXXX`

| Kode | Deskripsi               | Panjang |
|------|-------------------------|---------|
| TT   | 2 digit tahun masuk     | 2       |
| FF   | Kode fakultas           | 1       |
| PP   | Kode prodi              | 2       |
| J    | Kode jenjang            | 1       |
| S    | Status mahasiswa        | 1       |
| XXX  | Nomor urut              | 3       |

## Contoh Implementasi

| Komponen               | Nilai | Kode |
|------------------------|-------|------|
| Tahun masuk            | 2025  | 25   |
| Fakultas Teknologi Informasi | 1     | 1    |
| Prodi Teknik Informatika | 05   | 05   |
| Jenjang S1             | 2     | 2    |
| Status Reguler         | 1     | 1    |
| Nomor urut             | 005   | 005  |

**Hasil NIM:** `2510521005`
